## 主题：R2 vs S3

核心事实：
- R2 是 Cloudflare 的对象存储服务，最大特点是出口流量（egress）完全免费
- S3 是 AWS 的对象存储，出口按 GB 计费，每 GB 约 $0.09
- 两者 API 兼容（都用 S3 API），代码层可以直接用 aws-sdk 连 R2
- 存储成本：R2 $0.015/GB/月，S3 $0.023/GB/月（R2 略便宜）
- 中国大陆访问：R2 走 Cloudflare 全球节点，比 S3 us-east-1 快很多
- 适用场景：分发型场景（图床、视频、二进制文件下载）R2 完胜；纯归档（很少读）两者差不多
