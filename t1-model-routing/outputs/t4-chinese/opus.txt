独立开发者还在用 S3 存图床和视频的，赶紧算笔账。

R2 出口流量 0 费用，S3 每 GB 收 $0.09。你做个图床，月流出 1TB，S3 光出口就 $90，R2 这块是 0。存储那点钱反而是零头：R2 $0.015/GB/月，S3 $0.023/GB/月。

API 完全兼容，aws-sdk 改个 endpoint 就连上 R2，迁移成本基本为零。中国大陆用户体验 R2 也明显更好——Cloudflare 全球节点边缘命中，S3 us-east-1 那个延迟没法看。

适用判断很简单：
- 图床、视频、APK/二进制分发、任何"上传一次读 N 次"的场景:R2 完胜，没什么好犹豫的
- 纯冷归档(写多读少,基本不动):两边差距不大,看你账号在哪边顺手
- 已经深度绑定 AWS 全家桶(Lambda + S3 + CloudFront 联动):别为省这点钱拆生态

中小项目、个人项目、独立开发者,默认直接 R2,省下来的钱够付一年域名加 Pro 订阅。
