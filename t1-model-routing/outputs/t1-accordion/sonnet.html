Claude Code 使用指南

常见问题解答

Claude Code 是专为开发者设计的 CLI 工具，直接运行在终端和代码库中，能够读写文件、执行命令、操作 Git。ChatGPT 是通用对话 AI，主要在浏览器界面中使用。

  • Claude Code 可以直接访问你的文件系统和终端，ChatGPT 不行
  • Claude Code 支持 MCP（Model Context Protocol）扩展工具链
  • Claude Code 天然适合多文件重构、代码库导航、自动化任务

子代理（Subagent）允许主代理将子任务委托给独立的 agent 并行执行，提升复杂任务的处理效率。

  • 使用 Task 工具派发子任务，子代理在独立 context 中运行
  • 省略 subagent_type 默认为 fork，继承父进程缓存，成本最低
  • 子代理只负责执行和返回结构化摘要，决策权保留在主代理
  • 委托深度上限为 depth-1，子代理不再派生子代理

MCP（Model Context Protocol）是 Anthropic 推出的开放标准，允许将外部工具和数据源接入 Claude Code。常见支持的工具类型包括：

  • 浏览器自动化：Chrome DevTools、Playwright 等，支持截图、点击、表单填写
  • 云平台：Cloudflare（Pages、Workers、R2、D1）、GitHub 等
  • 知识库：Context7 文档查询、ChromaFS 向量文件系统
  • 搜索与抓取：Tavily 搜索、网页内容提取等

通过 /mcp 命令管理已连接的 MCP server，也可在 settings.json 中配置。
