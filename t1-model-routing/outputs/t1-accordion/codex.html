Claude Code 使用常见问题

快速了解 Claude Code 的工作方式、子代理能力，以及 MCP 工具扩展。

ChatGPT 更像通用对话助手，适合问答、写作和分析；Claude Code 更偏向本地开发执行环境，可以读取项目文件、运行命令、修改代码、验证结果，并围绕真实代码库完成端到端工程任务。
子代理通常用于拆分复杂任务，例如让一个代理负责代码审查，另一个代理负责实现或调研。你可以按任务目标分配上下文、约束和输出格式，让主流程汇总结果并决定下一步。
Claude Code 可以使用已配置的 MCP 服务器暴露出来的工具，例如浏览器自动化、文件或文档处理、数据库查询、设计工具、企业系统集成等。实际可用工具取决于当前工作区和本地配置。
