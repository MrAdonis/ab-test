# poster-cover — 营销海报/封面图生成

## When to Use / 不适用声明

**适用**：单张/系列营销图，含社交封面（X Article、小红书、公众号 Banner）、KV 主视觉、产品宣传图、活动海报。

**不适用，改用下方工具**：

| 场景 | 改用 |
|------|------|
| 结构化工程图（流程/架构/ER/时序） | `/diagram` |
| 数据统计图表（柱/线/饼/雷达） | `/diagram` 或 `/baoyu-infographic` |
| 需反复迭代的组件代码 | 项目框架组件化 |
| 交互演示稿 | `/prototype` |
| 纯文案排版（无设计需求） | 直接写 Markdown |

## 可测视觉规则（品味 → 阈值）

> 模式 10a：能量化的拆阈值并带依据；真无法量化的诚实留文字，不硬凑假数字。

### 一、留白（Whitespace）

**阈值：页面可感知留白面积 ≥ 30%**

依据：Calimari Rule——信息密度越低，感知档次越高。留白压缩到 <20% 时，视觉会自动把内容归类为"廉价堆砌"。

执行检验（HTML 渲染后目测）：遮住所有文字/图形，剩余空白区域能否覆盖整张图的 1/3？否则继续删减或扩大间距。

### 二、配色（Color Palette）

**主色 ≤ 2，辅助色（用于分隔/标注）≤ 1，共 ≤ 3 色**

依据：Cleveland & McGill 1984——人眼在一张图内可无歧义区分的颜色上限约为 5–7，但品牌/高端感场景主动收窄到 2–3 可消除"廉价感"的首要信号。

**文字与底色对比度 ≥ 7:1（正文）/ ≥ 4.5:1（大标题 ≥24px）**

依据：WCAG AAA（Web Content Accessibility Guidelines 2.1, Level AAA）。7:1 比率在强光/手机屏下仍可辨。

执行检验：用 CSS `color-contrast()` 或 WebAIM contrast checker 传入前景/背景 hex 值验证。

### 三、字重层级（Type Hierarchy）

**字重层级 ≤ 3 级**（例：Display 粗 → Body 常规 → Caption 细/斜）

依据：Mark Boulton《A Practical Guide to Designing for the Web》ch.4——超过 3 个字重层级时，读者无法快速建立视觉扫描路线，等价于层级消失。

**字号比例**：相邻层级字号比 ≥ 1.33x（Minor Third 音乐比例，视觉可感知差异的最小倍率）

**字体家族 ≤ 2 种**：标题 + 正文各一，不混第三种装饰字体。

### 四、视觉重心（无法量化，保留文字）

构图呼吸感无法用单一数值定义，以下是可观测的执行检查点：

- 主体（文字/图形）不居中塞满：避免主体铺满边缘，四边至少保留一处明显留白区
- 视觉重心偏上 1/3 或下 1/3（黄金比例变体）：正中央重心在静态图上显沉闷
- 文字块与图形之间保留明确的视觉分隔，而不是叠在一起靠透明度区分

## 工作流

### Step 1：需求采集

收集以下信息（缺即追问，不猜测）：

```
- 用途/平台（X Article 封面 / 小红书 / 公众号 Banner / 活动海报 / 其他）
- 核心文案（主标题、副标题、品牌/作者名，最多 3 层）
- 尺寸要求（若有；无则按平台默认，见下表）
- 主色调偏好（若无明确要求，给 3 个配色方向供选）
- 参考风格（若有图片链接或描述）
```

**平台默认尺寸（来自平台规范，免引用）**：

| 平台 | 尺寸 | 备注 |
|------|------|------|
| X Article 封面 | 1200×675px（16:9） | 推荐 |
| 小红书封面 | 1080×1350px（4:3 竖） | 主流比例 |
| 公众号 Banner | 900×383px | 文章头图 |
| 通用海报（A4 竖） | 794×1123px（96dpi）| 打印/数字两用 |

### Step 2：配色方案选定

根据品味规则生成 **3 套配色方案**（各含主色/辅色/背景色/文字色 hex 值），每套标注：
- 情绪/风格定位（"冷峻极简"/"温暖专业"/"克制活力"）
- 对比度验证结果（主文字 vs 背景的实际比率）

**输出格式示例**：

```
方案 A — 冷峻极简
背景: #F5F4F2  主文字: #0D0D0D  对比度: 17.1:1 ✅
主色: #1A1A2E  辅色: #E8D5B7  主色数: 2 ✅
```

等用户选定方案后再进入 Step 3。不要同时提交三套成品图。

### Step 3：HTML 草稿渲染

用 HTML + inline CSS 实现版式，交付物为 **PNG 截图**（不是 HTML 文件）。

**渲染检查清单（渲染前逐项过）**：

- [ ] 留白面积 ≥ 30%：移除所有装饰后，空白区域占比估算
- [ ] 主色 ≤ 2，辅助色 ≤ 1：数 CSS 颜色变量
- [ ] 对比度 ≥ 7:1（正文）/ ≥ 4.5:1（大标题）：代入 hex 计算
- [ ] 字重层级 ≤ 3，字族 ≤ 2
- [ ] 主体不居中塞满，四边至少一侧留白明显

**Good vs Bad（留白执行对照）**：

```css
/* Bad — 主体铺边到底，留白 < 15% */
.poster { padding: 16px; display: flex; flex-direction: column; gap: 8px; }
.title { font-size: 48px; margin: 0; }

/* Good — 显式留白层，确保 ≥ 30% 呼吸空间 */
.poster { padding: 80px 72px; display: flex; flex-direction: column; gap: 32px; }
.title { font-size: 48px; margin: 0; line-height: 1.1; }
/* 上下留白各占约 80/(总高*0.5)，保留至少 1/3 空白区 */
```

**Good vs Bad（配色层级对照）**：

```css
/* Bad — 四种颜色竞争，高级感消失 */
.title { color: #E63946; }
.subtitle { color: #2A9D8F; }
.tag { color: #E9C46A; background: #264653; }

/* Good — 主色 + 中性底，对比度锚定在文字本身 */
:root {
  --bg: #F5F4F2;
  --ink-primary: #0D0D0D;   /* 对比度 17:1 ✅ */
  --ink-secondary: #5C5C5C; /* 对比度 7.2:1 ✅ */
  --accent: #1A1A2E;        /* 仅用于装饰线/标签，≤1 辅色 */
}
```

### Step 4：截图交付与反馈

用 headless Chrome 渲染截图，resize ≤ 1280px 宽，JPEG quality 80。

交付格式：
1. PNG 图（主交付物）
2. 版规摘要（4 行以内）：实际留白估算 / 使用色数 / 对比度 / 字重层级数
3. 下一步选项：确认交付 / 调整配色方向 / 调整文案排布

## 约束

**文字渲染**：中文文字准确性要求高时，用 HTML 渲染（确定性）而非 AI 生图（不可控）。纯图形/无中文 → 可走 `codex-imagen.sh`（免费）或 `baoyu-imagine`（付费）。

**多方案对比**：Step 2 的配色方案最多 3 套；Step 3 之后每轮只改一个方向（配色 or 排版 or 文案，不同时改多个），方便归因。

**系列图**：同系列（如课程卡片、活动系列海报）Step 2 只选一次配色方案，后续所有张锁定同一套色板，不每张重新选色。

## Gotchas

> 只记真实踩过的坑（执行失败并定位过根因的），不记假设。

- **headless Chrome 中文字体回退乱码** → 根因：系统字体路径未注入 → 规避：HTML `<head>` 里显式引入 Google Fonts 或指定 `font-family: "PingFang SC", "Noto Sans SC", sans-serif`，截图前用 `document.fonts.ready` 等字体加载完成再 capture
- **对比度检验跳过导致交付物无障碍不合格** → 根因：凭肉眼"看起来够深"做判断 → 规避：Step 3 清单第三项强制计算，不允许目测过关
- **AI 生图中文文字乱打/错字** → 根因：扩散模型对汉字笔画学习不稳定 → 规避：凡含中文的海报一律走 HTML 渲染路径，不走图生图；图生图路径见 `~/.claude/references/image-base.md`
- **配色"克制感"方案反而发灰发脏** → 根因：低饱和配色未校正明度关系，相近明度的中性色叠在一起失去对比层次 → 规避：选低饱和配色时，前景/背景明度差 ≥ 50%（HSL Lightness 差值），仅靠色相区分不够

## 主观品味快查（无法量化部分）

以下特征无法折算为单一数值，执行时以**是/否检查**代替阈值：

| 品味目标 | 可观测检查点 |
|---------|-------------|
| 呼吸感 | 主体与留白之间没有元素"浮在中间"填补空隙 |
| 高级感 | 无渐变叠渐变、无描边叠描边，装饰元素总数 ≤ 2 |
| 配色克制 | 关掉色觉/灰度看，仍能区分信息层级（不依赖颜色传达层级） |
| 视觉重心稳 | 垂直方向：主要信息块在上 1/3 或中部，底部留白 ≥ 顶部留白 |
