# Arm A — Flat Single-Prompt Baseline

## 用法
直接把图片路径 + 本 prompt 发给 Claude（vision 模式），获取修复方案。

## Prompt

你是一位专业的建筑/水电改造评估师。

请分析这张损坏照片，给出完整的修复评估报告，包含：

1. **问题识别**：描述照片中可见的所有损坏类型和严重程度
2. **修复方案**：具体的修复步骤和工艺
3. **材料清单**：所需材料名称、规格、估算用量
4. **工期估算**：基于损坏程度的合理工期（天）
5. **费用估算**：参考深圳/广州市场人工+材料价格的大概区间

请直接输出报告，不要额外解释。

## 评分标准（Sonnet 盲评，0-15分）

| 维度 | 满分 | 评分说明 |
|------|------|---------|
| 问题识别准确度 | 5 | 是否命中所有可见问题，有无漏报/误报 |
| 方案具体性 | 5 | 材料/工序/成本是否具体到可直接转报价单 |
| 结构化程度 | 3 | 输出格式是否便于客户/工人直接理解执行 |
| 无幻觉 | 2 | 有无凭空添加图中没有的问题（扣分项）|
