关于 Claude Code 使用的快速解答

Claude Code 是 Anthropic 推出的命令行编程助手，专注于在终端里直接读写代码、执行命令、跑测试，可以理解整个代码库的上下文并完成多步骤工程任务。ChatGPT 主要是通用对话界面，虽然也能生成代码，但默认不直接操作本地文件系统，需要复制粘贴或额外插件。Claude Code 更像一个能动手的工程伙伴，ChatGPT 更像一个会答题的助手。
通过 Task 工具派发子代理处理独立任务，比如并行搜索、独立分析或代码审查。在 ~/.claude/agents/ 目录预定义专用 agent（如 researcher、code-reviewer），调用时指定 subagent_type 参数即可。省略该参数则走 fork 模式，继承主上下文和缓存，成本最低。子代理应只返回结构化摘要，不要返回原始输出，把判断权留给主代理。
MCP（Model Context Protocol）是开放协议，Claude Code 可接入任何符合规范的服务器。常见类别包括浏览器自动化（chrome-devtools、playwright）、云平台（Cloudflare、AWS）、文档检索（context7）、生产力工具（飞书、Notion、Gmail）、数据源（GitHub、数据库）和文件系统访问等。通过 claude mcp add 命令或编辑配置文件添加服务器，每个 MCP 提供的工具会自动注入到 Claude 的工具列表中可被直接调用。
