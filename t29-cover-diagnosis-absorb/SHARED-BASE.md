# 封面提示词框架（共用基底，A/B 两臂都拿到）

按视觉风格选框架。执行走 seedream（baoyu-imagine），prompt 写文件再 `--promptfiles` 传入。

## Style 1 — 弥散风（editorial diffuse gradient）
复古弥散渐变 + 编辑杂志排版 + 胶片颗粒 + 印刷噪点 + 高级留白。
适合：思考型/解析型文章，深度洞察但不需强冲击，偏编辑杂志审美。

## Style 2 — 巨型透视字（giant perspective typography）
巨型中文透视标题 + 高冲突撞色 + 主题叙事插图 + 强空间纵深 + 电影级构图。
适合：feed 强抓眼、概念性强、有核心主题词；中文标题本身视觉张力够。

## Style 3 — 极简黑白编辑风
黑白灰极简 + 文字主导 + 图像隐喻。三层文字系统：巨型主视觉词 → 完整标题 → 副标题。
适合：长文封面、概念表达、个人品牌；纯文字驱动想要高级沉稳感。

## Style 4 — 麦肯锡/咨询风
白/深蓝/黑/冷灰主色 + 视觉隐喻（漏斗/路径/阶梯/矩阵/坐标/飞轮等 15 种）+ 极简商业图形。
适合：商业分析/战略拆解；PPT 封面；要"咨询报告"质感。

## Style 5 — Nature/SCI 学术风
科研信息图 / 科研杂志封面 / 论文插图三套，Nature/Science/Cell 系克制配色。
适合：科研/医学/生命科学类内容；学术账号配图；论文 GA。

## Style 6 — 时装设计师手稿风（图生图，需 --ref 参考照片）
把人物照片重绘为手绘时装插画。仅英文标注。
适合：旅行/穿搭/生活方式配图；人像类封面；创意个人品牌。

## 选用判据
| 场景 | 选用 |
|------|------|
| 深度分析/思考型，偏沉稳 | Style 1 |
| 强概念 hook，feed 强抓眼 | Style 2 |
| 长文封面、概念表达、个人品牌 | Style 3 |
| 商业分析/战略拆解、PPT 封面 | Style 4 |
| 科研/医学/生命科学 | Style 5 |
| 有人物照片想做手绘风 | Style 6 |

## 平台适配
| 平台 | 画幅 | 推荐风格 |
|------|------|---------|
| X 文章封面 | 16:9 或 5:2 | 1/2/3/4 |
| 小红书封面 | 3:4 或 4:5 | 1/2/6 |
| 公众号封面 | 3:2 | 1/3/4/5 |
| PPT/商业报告 | 16:9 | 4 |
| 学术/科研 | 1:1 或 16:9 | 5 |
