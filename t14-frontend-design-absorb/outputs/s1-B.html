让 Git 管理
你的笔记版本

因为笔记是纯 Markdown 文件，你可以把整个 vault 当 Git 仓库。 分支写草稿，merge 确认，revert 找回。开发者最熟悉的工具，用在最重要的知识上。

  • 完整的 diff 历史，每次改动清晰可见
  • 多设备同步只需 git pull / push
  • 私有 repo 意味着笔记绝对私密
  • 支持任意外部编辑器同时编辑
